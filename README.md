1. 将训练数据与测试数据放置于 `datasets/round_2/训练数据` 与  `datasets/round_2/测试数据` 下（如目录结构所示）
2. 根据环境依赖配置环境（也有 requirements.txt 文件）
3. 确保 `submits/round_2/all_model_with_fi` 文件夹本地存在
4. 执行 `src/main_round2_final.ipynb`
